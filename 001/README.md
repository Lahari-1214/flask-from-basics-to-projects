<p>A framework is a pre-written structure that helps you build applications faster and in a standard way,
think like: A ready-made building structure where you just fill rooms instead of building from scratch.</p>

### Without framework (pure Python):
---

You must handle server

Routing

HTTP requests

Templates

Security

Sessions

Everything manually

### With framework like Flask:
---

Routing is ready (@app.route)

Templates ready (render_template)

Sessions ready

Request handling ready

You just write business logic.
