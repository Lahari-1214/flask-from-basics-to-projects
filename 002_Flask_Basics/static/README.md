1. Installing Flask: Before installing Flask, make sure you have Python installed on your system. You can check this by running `python --version` in your terminal. Once you have Python installed,then create a virtual environment as desribed in 2nd point, you can install Flask using pip, which is the package installer for Python. 
Run the following command in your terminal: pip install flask

2. Virtual environment: 
Run the following commands to create virtual environment:
    1. python -m venv venv
    2. venv\Scripts\activate

3. Basic app: observe the basic lines inside the app.py 

4. Running server:  Run the file app.py present in this folder.[python app.py]
